# SwiftGlobal Logistics

A professional logistics and freight company website with integrated real-time universal parcel tracking.

## Features
- Real-time parcel tracking (1,200+ carriers via TrackingMore API)
- Sea, Air, Land Freight service pages
- Customs Clearance, Warehousing, Project Cargo pages
- Fully responsive on all devices
- SEO optimized with sitemap and meta tags
- Smooth animations and modern design

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- TrackingMore API v4 (universal tracking)
- AOS Animation Library
- Font Awesome Icons
- Google Fonts (Inter + Poppins)
- Deployed on GitHub Pages
